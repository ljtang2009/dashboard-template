# dashboard-template

## dev

`npm run build:dev`

### 手动启动 api-server

`npm run build:dev manual-api-server`

### mock api-server

`npm run build:dev mock-api-server`

## build electron

### auto-platform

`npm run build:electron`

### build for Windows

`npm run build:electron platform=windows`

### build for Mac

`npm run build:electron platform=mac`
